# Dairy Milk Collection and Analysis System

This is a Python-based project for managing dairy milk collection, farmer records, payments, and reports.
It uses:
- Python (OOP concepts)
- MySQL for database
- NumPy for data analysis
- Matplotlib for data visualization

## Features
- Farmer management (Add, View, Update, Delete)
- Daily milk collection and payment calculation
- Reports and statistics
- Graphical visualization of milk data

## How to Run
1. Install required libraries:
   pip install mysql-connector-python numpy matplotlib

2. Create MySQL database and tables.

3. Set database credentials as environment variables:
   DB_USER and DB_PASSWORD

4. Run:
   python main.py
